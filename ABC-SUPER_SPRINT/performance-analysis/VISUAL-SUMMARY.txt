───────────────────────────────────────────────────────────────────────────────
  PHASE B: SIMPLE DRIVE SIMPLEX
───────────────────────────────────────────────────────────────────────────────

Drive System (Replaces Behavior State Machine):

  Old:  BehaviorMode enum (Catatonic, Wandering, Seeking, Fleeing)
        State transitions, wandering timers, random directions
        2.0ms per tick

  New:  DriveState component (continuous force accumulation)
        No states, no transitions, emergent behavior
        < 2.0ms per tick (target)

L1 Drive Computation:
  ┌─────────────────────────────────────────────────────────────────────────┐
  │ For each creature:                                                      │
  │   1. Scan L1 cells in perception range (~35 cells)                     │
  │   2. Accumulate repulsion from large crits (max_size > self)           │
  │   3. Accumulate attraction to empty space (total_mass < threshold)     │
  │   4. Write DriveState.direction (force vector)                         │
  │                                                                         │
  │ Complexity: 35 cells × 12 ops = 420 ops per creature                   │
  │ Estimate:   360K × 420 = 151M ops → 1.5ms (parallel, 16 cores)        │
  └─────────────────────────────────────────────────────────────────────────┘

Emergent Behaviors:
  ┌───────────────────────────────────────────┐
  │ Situation        │ Drive  │ Behavior      │
  ├──────────────────┼────────┼───────────────┤
  │ Empty L1 cell    │   0    │ Resting       │ (no code, just emergent)
  │ Crowded cell     │   ↓    │ Dispersing    │
  │ Large crit near  │   ↑    │ Fleeing       │
  │ Equilibrium      │   0    │ Resting       │
  └───────────────────────────────────────────┘

Performance Impact:
  ✓ Drive computation:  < 2ms (competitive with old behavior 2.0ms)
  ✓ IPC:                2.0+ (compute-bound, not memory-bound)
  ✓ L1D miss rate:      < 3% (BioSignatures are small, 8 bytes)
  ✓ Rayon parallel:     16 cores engaged
