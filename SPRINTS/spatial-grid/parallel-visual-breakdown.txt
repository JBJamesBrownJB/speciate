==============================================================================
MEMORY LAYOUT (Cache Line Alignment)
==============================================================================

WITHOUT ALIGNMENT (Potential False Sharing)
┌────────────────────────────────────────────────────────┐
│ Cache Line 0 (64 bytes)                                │
│ ┌──────────┬──────────┬──────────┬──────────┐          │
│ │Cell[0]   │Cell[1]   │Cell[2]   │Cell[3]   │ ⚠️       │
│ │offset:u32│offset:u32│offset:u32│offset:u32│ False   │
│ │counter:  │counter:  │counter:  │counter:  │ Sharing │
│ │AtomicU32 │AtomicU32 │AtomicU32 │AtomicU32 │ Risk!   │
│ └──────────┴──────────┴──────────┴──────────┘          │
└────────────────────────────────────────────────────────┘
If Thread 1 writes Cell[0] and Thread 2 writes Cell[1]
→ Same cache line → Cache ping-pong

WITH ALIGNMENT (No False Sharing)
┌────────────────────────────────────────────────────────┐
│ Cache Line 0 (64 bytes)                                │
│ ┌──────────────────────────────────────────────────┐   │
│ │Cell[0]: offset: u32, counter: AtomicU32, padding│ ✓  │
│ └──────────────────────────────────────────────────┘   │
└────────────────────────────────────────────────────────┘
┌────────────────────────────────────────────────────────┐
│ Cache Line 1 (64 bytes)                                │
│ ┌──────────────────────────────────────────────────┐   │
│ │Cell[1]: offset: u32, counter: AtomicU32, padding│ ✓  │
│ └──────────────────────────────────────────────────┘   │
└────────────────────────────────────────────────────────┘

Implementation:
#[repr(align(64))]
struct AlignedAtomicCell {
    offset: u32,
    counter: AtomicU32,
    _padding: [u8; 56], // Pad to 64 bytes
}

Cost: 2000 cells x 64 bytes = 128KB (acceptable)
