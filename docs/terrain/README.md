**Category:** 💡 **Ideas** — brainstormed, **not committed**, **not implemented**. The simulation currently uses hardcoded static terrain; everything here is exploratory.

**Purpose:** Concept notes for a dynamic, evolving world that acts as a fitness test — terrain creatures shape and adapt to, rather than a static backdrop. Ties into the DNA-driven mandate via habitat preference genes.

**What's inside:**

| File | What it explores |
|------|------------------|
| [`cellular-automata-terrain.md`](cellular-automata-terrain.md) | Grid-based CA world (grass growth, fire spread, water flow) driving boom-bust ecology and carrying capacity. Chunked/sparse-storage notes for 2B-tile scale. |
| [`habitat-hints.md`](habitat-hints.md) | Habitats nudge offspring toward beneficial genes, so a species "reacts" to a cultivated niche over a few generations. Possible LLM-assisted hint calibration. |
| [`concepts.md`](concepts.md) | Raw list of candidate natural features (dunes, algal mats, stromatolites, tundra, meadow, volcanic, steppe). |

**Status NOW?** None of this is scheduled. The active NOW pillars are [`docs/scale/`](../scale/) and [`docs/visuals/`](../visuals/) — see [`docs/ROADMAP.md`](../ROADMAP.md) for the authoritative NOW / NEXT / DREAM tiers.
